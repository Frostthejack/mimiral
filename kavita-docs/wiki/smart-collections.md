MAL Stack Import (and sync) Kavita+ allows any user to import their MAL stacks and restacks from their account into Kavita. Kavita will build a collection that syncs every 2 days with the upstream MAL stack. Kavita will sync once every 2 days to MAL. This period is not adjustable. Want to Read sync Kavita supports the ability to add Series to Want to Read from a user’s MAL/AniList Planned to Read list.
